URL Shortener FCC

Pass in a url that you want to shorten:

https://not-so-short-url.herokuapp.com/new/https://google.com

And you will receive it "shortened":

{ "original_url":"https://google.com", "short_url":"https://not-so-short-url.herokuapp.com/28759" }

Then you can navigate to:

https://not-so-short-url.herokuapp.com/28759

And you will get redirected to:

https://google.com
